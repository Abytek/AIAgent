# AIAgent

## Installation

1. **Install the package globally**
   ```powershell
   npm install -g
   ```
   This registers the abytek-aiagent-cli command.

2. **Start the 9router UI**
   ```powershell
   abytek-aiagent-cli 9router
   ```
   Follow the prompts in the UI to set up 9router and add a combo named default.

3. **Run the root**
   ```powershell
   abytek-aiagent-cli root.default
   ```

4. **Run the runtime**
   ```powershell
   abytek-aiagent-cli runtime.default
   ```
